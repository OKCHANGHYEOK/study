* 반복문
- 특정 작업을 여러번 반복하여 사용하는 구조
- 조건이 정해져있다.
- for, while, do-while 의 세가지 종류가 있다.

* for 문
- for ( 초기식; 조건식; 증감식 ) { 반복할 코드 }
- 초기식, 조건식, 증감식의 형태로 구성된 반복문
- 초기식 : int a = 1;
- 조건식 : a < 5;
- 증감식 : a++

* while 문
- while (조건식) { 반복할 코드 }
- 반복의 종료가 존재하는 경우와 존재하지 않는 경우가 있다.
- while 내부에 특정 조건일 때만 반복하는 경우 -> 조건이 맞지 않게되면 종료
- while 내부에 항상 참인 조건이 존재하는 경우 -> 반복이 종료되지 않는다.

* do-while 문
- 최소 한 번은 반드시 실행되어야 하는 반복이 필요할 때 사용
- do { 반복할 코드 } while (조건식)
