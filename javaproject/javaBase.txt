* 클래스
 - 자바의 기능들의 집합
 - 모든 자바 코드는 클래스 내부에 작성된다.
 - 메서드를 정의하고 호출하는 것들을 묶어주는 것

* 메인 함수
 - public static void main(String args[]) 로 선언되어 있는 함수이다.
 - 자바의 함수는 항상 이 메인 함수 내부에서 실행된다. = 자바의 프로그램의 시작점

* 변수
 - 값을 담는 공간을 가진 상자
 - 값 : 숫자, 문자, 불린
 - 자바에서의 변수 선언은 '자료형 변수명' 형식으로 하여야한다.
 - 숫자 자료형에는 int, float, double 등이 있다.
  -> int : 정수 / int a = 1;
  -> float : 실수 / float b = 1.1f;
  -> double : 실수 (범위가 float 보다 넓다.) / double c = 2.22d;
 - 문자 자료형에는 char 가 있다. / char d = '오';
 - 불린 자료형에는 boolean 이 있다. / boolean e = true;
